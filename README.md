# Organizador Automático de Arquivos em Python

Script em Python que organiza arquivos automaticamente em pastas
com base em suas extensões.

## Funcionalidades
- Organiza arquivos .txt, .pdf, .jpg e .png
- Cria pastas automaticamente
- Move arquivos para seus respectivos diretórios
- Arquivos desconhecidos vão para a pasta "outros"

## Como usar
1. Crie uma pasta chamada `organizador`
2. Coloque os arquivos dentro dela
3. Execute o script:

```bash
python organizador.py
